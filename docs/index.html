This console talks to a live, public emulator of the Mobile Industrial Robots REST API running on AWS Lambda. Each tracked MiR software generation is mounted at its own path prefix — /<version>/api/v2.0.0/… — so you can diff robot behavior across releases from one URL. State (mission queue, PLC registers, status writes) is in-memory per runtime instance and resets on cold start. Auth mirrors a factory robot: Basic BASE64(user:SHA-256(password)), default account distributor / distributor.
