The official MiR robot API docs

The REST API reference on MiR's support portal — one PDF per version, behind a free account. The primary source for every spec used here.

Endpoints

  

    
PathWhat it serves

    
/consoleThis page (also served at / to browsers) — catalog, telemetry, virtual robot sessions.

    
/<version>/api/v2.0.0/…The emulated MiR robot REST API for that software version (also /latest/…).

    
/fleet/<version>/api/v1/…The emulated MiR Fleet Enterprise Integration API (also /fleet/latest/…); auth is an x-api-key header, default key distributor. Each fleet embeds robot emulators it controls over their own REST API.

    
/fleet/<version>/openapi.jsonThe official Fleet OpenAPI 3 document, served verbatim.

    
/<version>/swagger.jsonMachine-readable spec, Swagger 2.0 (and /<version>/openapi.json for OpenAPI 3.0).

    
/<version>/Per-version JSON index: operation count, auth, session header, spec provenance.

    
/JSON index of everything above — browsers get this page instead.

    
/healthzLiveness + the list of tracked versions.
